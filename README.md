# simple-scientific-calculator project

## Overview

simple-scientific-calculator is a desktop calculator application developed using Python and Tkinter. It combines multiple tools in a single application, including scientific calculations, unit conversions, graph plotting, memory functions, and calculation history management.

The project is designed with a user-friendly interface and supports both basic and advanced mathematical operations.

## Features

### Basic Calculator

* Addition, subtraction, multiplication, and division
* Percentage calculations
* Power and square root operations
* Parentheses support
* Answer recall (Ans)

### Scientific Functions

* Trigonometric functions

  * sin
  * cos
  * tan
  * asin
  * acos
  * atan

* Hyperbolic functions

  * sinh
  * cosh
  * tanh
  * asinh
  * acosh
  * atanh

* Logarithmic functions

  * ln
  * log
  * log10
  * log2

* Exponential functions

  * exp
  * powers
  * roots

### Angle Modes

* Degree (DEG)
* Radian (RAD)
* Gradian (GRAD)

### Unit Converter

Supports multiple categories:

* Length
* Mass
* Temperature
* Area
* Volume
* Speed
* Time

### Graph Plotter

* Plot mathematical functions
* Custom x-range
* Adjustable number of points
* Uses Matplotlib and NumPy

### Memory Functions

* MC (Memory Clear)
* MR (Memory Recall)
* M+ (Memory Add)
* M- (Memory Subtract)

### History Management

* Stores previous calculations
* Timestamp for each calculation
* Export history to a text file
* Reload expressions from history

### User Interface

* Dark Theme
* Light Theme
* Responsive button sizing
* Keyboard shortcuts
* Copy and Paste support

## Technologies Used

* Python
* Tkinter
* NumPy
* Matplotlib

### Install Required Libraries

pip install numpy matplotlib

### Run Application

python simple_scientific_calculator.py
